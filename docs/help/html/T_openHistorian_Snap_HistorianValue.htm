      Fields

      Value1 Value 1 should be where the first 64 bits of the field is stored. For 32 bit values, use this field only.
      Value2 Should only be used if value cannot be entirely stored in Value1. Compression penalty occurs when using this field.
      Value3 Should contain any kind of digital data such as Quality. Compression penalty occurs when used for any other type of field.
